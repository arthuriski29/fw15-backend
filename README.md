# How to use (Locally)

## 1. Clone the project

```bash
  git clone https://github.com/arthuriski29/fw15-frontend
```

## 2. Go to the project directory

```bash
  cd fw15-frontend
```

## 3. Install dependencies

```bash
  npm install
```

## 5. Start debugging and it will automatically installed debug in your phone

```bash
  npm run dev
```
run it in your browser :)
